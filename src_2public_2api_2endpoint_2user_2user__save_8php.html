Detailed Description

Save a user.

  • The operation is permitted if
    • The caller is in the group admin and pass is NULL or unset in the request, ie. the caller is not trying to set the password of the user. This prevents the admin from taking over an account.
    • The caller is the user that's being saved and groups is NULL, unset or equal to the current user groups, ie. the caller is not trying to set groups. This prevents privilege escalation.
HTTP Method
POST
Authentication
By token
Allowed groups
ALL
Rate limiting
This endpoint is rate limited.
Parameters
application/json
  • string user The name of the user to save. [required]
  • string|NULL pass The new password of the user. [optional]
  • array|NULL groups The new groups of the user. [optional]
Response
application/json
  • object user
    • string name The name of the saved user.
    • array groups The new groups of the user.
Status
  • 200 On success
  • 400 If the request parameters are invalid.
  • 401 If the caller is not allowed to save the user data. See above.
  • 404 If the user is not found.
