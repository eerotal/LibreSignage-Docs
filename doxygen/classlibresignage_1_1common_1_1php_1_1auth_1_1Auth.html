Ensure a user is authenticated and has permission to access a resource. This function is intended to be called at the beginning of web interface pages to only grant access for specific users and/or groups.

Parameters
array$user_wlAn array of whitelisted users.
array$group_wlAn array of whitelisted groups.
bool$redirIf TRUE, clients are redirected to error pages if they don't have permission to access a page.
string$tokenAn authentication token to use. If left unspecified, the token is read from the Auth::TOKEN_COOKIE cookie.
