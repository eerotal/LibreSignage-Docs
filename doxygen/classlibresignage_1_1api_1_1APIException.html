Detailed Description

APIException class for sending error responses to clients. This class can either be instantiated directly or you can rely on the global error handler to convert all exceptions to APIExceptions. If you're writing code for the API system itself, try to stick to directly creating APIExceptions though, since it's a bit more flexible. Call APIException::setup() to setup the global error handler.
