Class: ValidatorSelector

libresignage/ui/validator.ValidatorSelector(validate, style, validators, hooks)

A selector class for selecting input elements to validate.

Constructor

new ValidatorSelector(validate, style, validators, hooks)

Construct a ValidatorSelector object. The 'style' HTML DOM element should be a DIV element that contains the input element 'validate'. The 'style' DIV can also contain another DIV with the class 'invalid-feedback'. The invalid input message is written into this DIV. 'style' and 'validators' can be left null if they aren't needed.
Parameters:
Name Type Description
validate object The HTML DOM element to validate
style object | null The HTML DOM element to style.
validators Array.<BaseValidator> | null An array of validator objects.
hooks Array.<ValidatorSelector~state_change_hook> An array of hooks.
